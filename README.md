# Retos de Programación en Java

Los retos se encuentran dentro de la carpeta **Loops/src** en el archivo **Retos.java**.  
Para poder ejecutar cada reto, es importante revisar que todo el código está comentado y organizado en bloques, de manera que sea más fácil de interpretar.  

## Ejecución de los retos
- Abre el archivo `Retos.java`.  
- Localiza el bloque del reto que deseas ejecutar.  
- **Descomenta** el bloque correspondiente (quitando los símbolos de comentario).  
- Ejecuta el programa en tu IDE.  

⚠️ Nota: Solo debe estar **descomentado un reto a la vez** para evitar conflictos en la ejecución.
